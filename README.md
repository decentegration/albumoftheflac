# albumoftheflac

`albumoftheflac` — это Python CLI-утилита для автоматического обновления метаданных FLAC-альбомов. Она читает теги `ARTIST` и `ALBUM` из локальных файлов, ищет релиз на [Album of the Year](https://www.albumoftheyear.org/), извлекает жанры и дату релиза, а затем записывает их обратно во все `.flac`-файлы папки.

Проект решает простую, но практическую задачу: убрать ручное редактирование музыкальной библиотеки и автоматически подтягивать недостающие теги из внешнего источника.

## О проекте

### Задача

Сделать консольный инструмент, который:

- читает базовые теги альбома из FLAC-файлов;
- ищет нужный релиз на Album of the Year;
- парсит HTML без официального API;
- нормализует дату релиза;
- записывает `GENRE` и `ORIGINALDATE` обратно в аудиофайлы.

### Что реализовано

- CLI-входная точка `aotf`;
- рекурсивная и нерекурсивная обработка папок;
- парсинг страницы поиска и страницы альбома на Album of the Year;
- работа с FLAC-тегами через `metaflac`;
- нормализация даты в форматы `YYYY`, `YYYY-MM` или `YYYY-MM-DD`;
- упрощение отдельных жанров до более общих категорий.

### Что показывает этот проект в портфолио

- умение делать прикладные CLI-инструменты;
- работу с внешним сайтом без API;
- обработку полуструктурированных данных;
- интеграцию Python-кода с системными утилитами;
- автоматизацию реального пользовательского сценария.

## Как это работает

1. Утилита находит директории с `.flac`-файлами.
2. Берёт теги из первого файла альбома.
3. Извлекает `ARTIST` и `ALBUM`.
4. Формирует поисковый запрос для Album of the Year.
5. Открывает найденную страницу релиза.
6. Извлекает жанры и дату выхода из HTML.
7. Приводит дату к единому формату.
8. Записывает обновлённые теги во все FLAC-файлы директории.

## Установка

### Системная зависимость

Для работы нужен `metaflac`, доступный в `PATH`.

Пример для Debian/Ubuntu:

```bash
sudo apt install flac
```

### Python-зависимости

При использовании `uv`:

```bash
uv sync
```

Если нужен обычный `pip`:

```bash
pip install .
```

## Запуск

Рекурсивная обработка вложенных папок:

```bash
uv run aotf /path/to/music
```

Обработка только одной директории:

```bash
uv run aotf /path/to/album --non-recursive
```

Короткая форма:

```bash
uv run aotf /path/to/album -n
```

## Пример сценария

Если в FLAC-файлах уже есть теги:

- `ARTIST=Kendrick Lamar`
- `ALBUM=To Pimp a Butterfly`

после запуска утилита может проставить, например:

- `GENRE=Hip Hop, Jazz Rap`
- `ORIGINALDATE=2015-03-15`

## Структура проекта

```text
sources/
└── albumoftheflac/
    ├── aotf.py             # CLI entry point
    ├── orchestrator.py     # основной сценарий обработки
    ├── aoty_parsing.py     # парсинг Album of the Year
    ├── tags.py             # чтение и запись FLAC-тегов
    ├── date.py             # нормализация даты
    ├── simplify_genres.py  # упрощение жанров
    └── text.py             # текстовые утилиты
```

## Ограничения

- Парсер зависит от текущей HTML-структуры Album of the Year.
- Сейчас используется первый найденный результат поиска.
- Для работы нужны уже заполненные теги `ARTIST` и `ALBUM`.
- Во время обработки нужен доступ в интернет.
